# Android Security Research Lab (ASRL)

Android Security Research Lab (ASRL) 是一个面向 Android Framework、安全研究和漏洞分析的长期实验平台。

本项目并不是 AOSP 的镜像，而是围绕 Android 安全研究建立的一套工作流，包括：

- AOSP Workspace 管理
- Framework 编译
- Emulator 调试
- CVE 分析
- 0day 研究
- PoC / EXP 管理
- Android 知识库
- 自动化脚本
