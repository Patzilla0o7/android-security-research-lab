## lab 脚本说明

`lab` 脚本位于 `bin/lab.sh`，是统一的命令入口，负责将子命令分发到对应处理逻辑。主要行为：

- **横幅与帮助**：命令无参或使用 `help` 时会打印项目横幅与使用说明。
- **版本**：`version` 命令会从仓库根目录的 `VERSION` 文件读取版本并输出；若文件不存在则显示 `Unknown`。
- **支持的命令**：当前脚本列出的子命令包括 `help`、`version`、`doctor`、`bootstrap`、`workspace`、`repo`、`build`、`research`。
- **未实现的命令**：除 `help` 与 `version` 外的大多数命令目前输出 `Not Implemented Yet`，代表这些功能需要在其他位置实现具体逻辑。
- **未知命令处理**：传入未识别的子命令会打印帮助并以非零状态退出。

示例：

- `lab help` — 显示帮助
- `lab version` — 显示项目名与版本

注意：`bin/` 目录应保持为命令分发与入口实现，具体业务逻辑请在其他脚本或模块中实现并由 `lab` 调用。
